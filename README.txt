Mie scattering Analytic Solution

VS2008で作成したものです。

TM モードの部分は岡田さんが作ったもので、
DLL_MIE.dllとDLL_MIE.libというライブラリになっています。

TE モードの計算は MIE.h ファイルの中にあります。
上の部分にある ITERATION 変数の値を高くすると
精度が高くなるが、計算時間が長くなります。

main.cpp の中身は簡単に説明すると

keyboard() 関数：キーを押すと表示が変わったり、
指定した半径の円上の強度をファイルに出力します。
'q'を押すと終了します。

main()関数：基本的にこの中でTM, TEモードを変えたり、
散乱体の半径、屈折率、波長などのパラメータの設定します。
Mie散乱かWGMかはコメントを外して選択できます。
WGM のシミュレーションはいくつかのモードで
CMode, RMode 変数を使って簡単に設定できるようにしました。
